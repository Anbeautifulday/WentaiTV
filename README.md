# WentaiTV

### 实现功能：列表展示各种数据，包括头像，首帧
### 点击列表播放，播放控制，暂停，上一、下一曲，
### 下载，自动调整横竖屏幕，保持进度

#### 开发环境：Ubuntu 16.04 LTS
#### Android Studio 2.3
